Tools Used

Github Repository - Magpies

GitHub

GitHub Repository was used to store the documents and media items that were used for the website. The website was created using Visual Studio Code linked to GitHub Desktop. The members of the group were granted contributor access so that each member of the group had access to view and update code. This process worked quite well for the team members to review the content, given we had the team spread across Australia in different time zones. Due to time zones and access to team members, Word was used by the team to draft the deliverables for the website and later entered in GitHub. Hence the GitHub audit log may not accurately represent the work done by all within the team.

Microsoft Teams - Magpies

Microsoft Teams

Microsoft Teams was the tool chosen by the group as our repository for storage of documents, booking meetings, and chat amongst the group members. It also served as a calendar for reminders. Meetings were organised through Teams and each meeting had a designated person for taking minutes. The meetings were recorded so we could go back to them to view and listen to what we discussed. Minutes of the meetings are taken by a team member and posted to the folder in Teams. Teams served the group as a collaboration tool to discuss issues and get peer reviews of the work assigned.
